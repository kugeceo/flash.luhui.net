滚动条配色工具可以帮助您快速设计适合您网站风格的滚动条样式，并自动生成相应的CSS代码。

完成设计后，只需点击"复制CSS"按钮，即可将生成的代码应用到您的项目中。

这个工具支持所有现代浏览器，包括Chrome、Firefox、Edge和Safari。

底部内容区域

生成的CSS代码

/* 自定义滚动条样式 */
::-webkit-scrollbar {
  width: 8px;
}

/* 滚动条轨道 */
::-webkit-scrollbar-track {
  background: #f1f1f1;
  border-radius: 4px;
}

/* 滚动条滑块 */
::-webkit-scrollbar-thumb {
  background: #c1c1c1;
  border-radius: 4px;
}

/* 滚动条滑块悬停状态 */
::-webkit-scrollbar-thumb:hover {
  background: #a8a8a8;
}

/* 滚动条滑块激活状态 */
::-webkit-scrollbar-thumb:active {
  background: #8d8d8d;
}

/* Firefox 支持 */
* {
  scrollbar-width: thin;
  scrollbar-color: #c1c1c1 #f1f1f1;
}

使用说明

1. 使用左侧控制面板调整滚动条的宽度、圆角和各种颜色参数

2. 在预览区域实时查看滚动条效果

3. 满意后点击"复制CSS"按钮将代码复制到剪贴板

4. 将复制的CSS代码粘贴到您的项目样式表中

提示：此工具生成的代码兼容Chrome、Firefox、Edge和Safari等现代浏览器
