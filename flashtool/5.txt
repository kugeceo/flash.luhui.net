Flash MX / Flash MX 2004 滚动条配色 

在 Flash MX/2004 中，滚动条配色因使用的组件版本而异：老版 ScrollBar 用 setStyle 改色，新版 UI ScrollBar 需改皮肤或用 themeColor 整体换肤。

---

### 快速选择
- 组件面板里是“ScrollBar”（MX 时代旧组件）→ 用 setStyle 逐个改色（更灵活）。
- 组件面板里是“UI ScrollBar”（MX 2004 新组件）→ 用 themeColor 整体换肤，或直接编辑皮肤里的形状填充。

---

### 老版 ScrollBar（MX 及 2004 旧组件）：用 setStyle 改色
将下列代码中的 sbar 替换为你的滚动条实例名，按需改色。

```as
sbar.setStyleProperty("arrow", 0xff0000);       // 箭头
sbar.setStyleProperty("highlight", 0xffffff);    // 左上内高光
sbar.setStyleProperty("highlight3D", 0xff0000);  // 左上外高光
sbar.setStyleProperty("shadow", 0xff99cc);       // 右下内阴影
sbar.setStyleProperty("darkShadow", 0xff0000);   // 右下外阴影
sbar.setStyleProperty("scrollTrack", 0xcc6666);  // 轨道
sbar.setStyleProperty("face", 0xffcccc);         // 滑块与两端按钮
```

---

### 新版 UI ScrollBar（MX 2004 新组件）
- 方法A：用 themeColor 整体换肤（AS2）🎨
在第一帧或组件初始化处执行，会影响所有使用该主题的组件。

```as
Style.setStyle("themeColor", 0x5255b2);
```

- 方法B：手动改皮肤（彻底自定义）🧩
  1. 双击舞台上的 UI ScrollBar 进入皮肤编辑。
  2. 选择箭头/滑块等形状，在属性面板改填充色。
  3. 若整体风格未变，检查是否被 themeColor 覆盖，必要时移除或调整。

---

### 常见问题与排错
- 改了颜色没生效：确认组件版本；UI ScrollBar 优先受 themeColor 影响，或需编辑皮肤。
- 个别组件不想被 themeColor 影响：对该组件单独设样式，或改用老版 ScrollBar。
